# Masterplan

## Purpose

Build a small internal game that helps adults learn, practice, and refine reading time on an analog clock.

## Users and Flow

Players are everyday adults who want a quick practice game for analog clock reading. They open the game, choose a practice mode, play a 10-round session, and see how well they did through score, time taken, and streak.

The game has two modes:

- Read the clock: the game shows an analog clock and the player enters the time in 12-hour format.
- Set the clock: the game gives a time and the player sets the clock hands using sliders.

Each answer gives immediate feedback. Wrong answers are counted wrong and the game moves to the next round.

## Features

- Game shell and mode choice: to build. Done when a player can open the game, choose either "read the clock" or "set the clock," and begin a session.
- 10-round session engine: to build. Done when the game generates 10 random 5-minute-increment times, tracks the current round, and ends after round 10.
- Read the clock mode: to build. Done when the game shows an analog clock and accepts a typed 12-hour answer for each round.
- Immediate answer feedback: to build. Done when each submitted answer is marked correct or wrong, wrong answers move to the next round, and the correct time is shown.
- Set the clock mode: to build. Done when the game shows a written time and the player can set the analog clock hands using keyboard-accessible sliders.
- Score, time, streak, and pass result: to build. Done when the end screen shows score, completion time, best streak, and pass/fail with passing only at 10 out of 10.
- High-contrast keyboard-ready polish: to build. Done when the full game can be completed without a mouse and remains readable in high contrast.

## Data

The first version uses no saved or external data. Times are generated by the game in 5-minute increments.

## Access

The first version is for a small internal team.

## Quality Bar

- A player can complete a full 10-round session in either mode.
- Every generated time is on a 5-minute increment.
- A correct answer is accepted in 12-hour format.
- A wrong answer is counted wrong and advances to the next round.
- The game shows score, time taken, and streak by the end of a session.
- The result clearly says whether the player passed or failed.
- The player passes only by answering all 10 rounds correctly.
- The game can be played without a mouse.
- Text and clock controls remain readable in high contrast.

## Out of Scope

- User accounts.
- Saved progress.
- Leaderboards.
- External data or APIs.
- AM/PM training.
- Minute-by-minute difficulty.

## Edge Cases

- Empty typed answer.
- Invalid typed answer.
- Time entered without a leading zero.
- Slider value between valid 5-minute increments.
- Final round answer and end-of-game result.
- Losing keyboard focus during play.
